# Elastic Certified SIEM Analyst Exam Prep

Resources and study materials for the Elastic Certified SIEM Analyst certification exam.

## Overview

This repository contains notes, practice exercises, and reference materials for preparing for the Elastic Certified SIEM Analyst exam.

## Exam Topics

- Security analytics with Elastic Security
- Detection rules and alerts
- Threat hunting and investigation
- SIEM data ingestion and normalization
- Dashboards and visualizations for security monitoring
- Incident response workflows
